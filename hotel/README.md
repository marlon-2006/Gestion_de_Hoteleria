# Sistema de Gestión Hotel JW Marriott Quito

Proyecto Java Swing para NetBeans.

## Inicio de sesión

- Administrador: `ADM001`
- Recepcionista: `REC001`
- Housekeeper: `HK001`

## Persistencia de datos

La información se guarda mediante archivos de texto dentro de la carpeta `datos`:

- `habitaciones.txt`
- `empleados.txt`
- `clientes.txt`
- `reservas.txt`

Los archivos se crean automáticamente la primera vez que se ejecuta el programa y se actualizan al guardar cambios o cerrar la aplicación.

## Habitaciones iniciales

- Simples: 101 a 110
- Dobles: 201 a 208
- Suites ejecutivas: 301 a 304
- Suites presidenciales: 401 a 402

## Ejecución

Abrir el proyecto en NetBeans y ejecutar `main.Main`.
