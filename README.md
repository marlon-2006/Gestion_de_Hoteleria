## Descripción

Este proyecto consiste en un sistema de gestión hotelera desarrollado en Java utilizando Programación Orientada a Objetos y Swing como interfaz gráfica.

El sistema permite administrar clientes, habitaciones, reservas y empleados de un hotel, además de generar facturas y almacenar la información mediante archivos de texto (.txt).

## Funcionalidades

- Registro de clientes.
- Consulta de habitaciones disponibles.
- Registro de reservas.
- Consulta de reservas.
- Gestión de empleados:
  - Administrador.
  - Recepcionista.
  - Housekeeper.
- Generación de facturas.
- Persistencia de datos mediante archivos TXT.

## Persistencia

La información del sistema se almacena mediante archivos de texto (.txt), permitiendo conservar los datos de clientes y reservas entre ejecuciones del programa.
