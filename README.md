# Sistema de Gestión Hotelera - Hotel JW Marriott Quito

## Integrantes
- Marlon Ruben

## Descripción
Proyecto desarrollado en Java utilizando Programación Orientada a Objetos y Swing.

## Funcionalidades
- Registro de clientes
- Registro de reservas
- Consulta de habitaciones disponibles
- Gestión de empleados
- Facturación
- Persistencia de datos mediante archivos TXT

## Tecnologías
- Java
- Apache NetBeans
- Swing
- Git
- GitHub
